{# Time-of-Day Chip ================ Selects a time of day. The chip is self-contained: when no value is set it displays "at a specific time of day" as its own placeholder, so it can sit alone in an optional clause without external lead text. Params: - name: Form field name (default: "time") - value: Current value — preset word ("morning", "afternoon", "evening", "night") OR HH:MM (e.g. "14:30"). Empty string means unconfigured. Display rules: - empty -> "at a specific time of day" (placeholder) - preset word -> the word itself ("morning") - HH:MM time -> "at H:MM am/pm" (e.g. "at 2:30 pm") Presets map to wall-clock times in the cron processor: morning=00:00, afternoon=12:00, evening=17:00, night=23:59 #} {% set name = name or 'time' %} {% set value = value if value is defined else '' %} {% set variant = 'time' %} {% set quick_picks = ['morning', 'afternoon', 'evening', 'night'] %} {% set preset_displays = { 'morning': 'in the morning', 'afternoon': 'in the afternoon', 'evening': 'in the evening', 'night': 'at night' } %} {% set is_preset = value in quick_picks %} {% set is_custom = (':' in value) %} {% set is_placeholder = not value %} {% if is_placeholder %} {% set display = 'at a specific time of day' %} {% elif is_preset %} {% set display = preset_displays[value] %} {% else %} {% set _h = value.split(':')[0]|int %} {% set _m = value.split(':')[1] %} {% set _period = 'pm' if _h >= 12 else 'am' %} {% set _h12 = (_h % 12) if (_h % 12) else 12 %} {% set display = 'at ' ~ _h12 ~ ':' ~ _m ~ ' ' ~ _period %} {% endif %}
