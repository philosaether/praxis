Clause-Based Sentence

Required clauses (every…, create…) sit at the root level. Optional clauses nest beneath their parent, prefixed with ↳ when inactive. Click the ↳ to activate; the glyph becomes × so you can deactivate again.

Schedule + Task with Optional Time-of-Day

{# Required clause: every... #}
Every {% with name="clause_interval", value="weekdays", show_day_chip=false %} {% include "partials/chips/chip_interval.html" %} {% endwith %} ,
{# Optional clause: time-of-day chip (no lead text — chip is self-contained) #}
{% with name="clause_time", value="" %} {% include "partials/chips/chip_time.html" %} {% endwith %} ,
{# Scaffold clause: when [conditions are met] — visual preview only #}
when conditions are met ,
{# Required clause: create a task named [name] #}
create a task named {% with name="clause_task_name" %} {% include "partials/chips/chip_task_name.html" %} {% endwith %} ,
{# Optional clause: due [end of day] #}
due end of day {% with name="clause_due" %} {% include "partials/chips/chip_due.html" %} {% endwith %} ,
{# Optional clause: tagged [tags] #}
tagged with one or more tags {% with name="clause_tags" %} {% include "partials/chips/chip_tags.html" %} {% endwith %} ,
{# Scaffold clause: and notify [me] — visual preview only #}
and notify me ,
